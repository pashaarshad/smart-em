Fest Management and Online Registration Website

A Synopsis for "Smart Event Management System"

1. Abstract

This project is developed to make the management of college fests easier and more organized. During fests, handling multiple events, participant registrations, and payments manually becomes difficult and time-consuming. To solve this problem, a web-based Fest Management Website is designed where all fest-related activities can be managed from a single platform.

The website allows users to view all fest events, register for events online, and make payments. The frontend of the application is built using React and Next.js to provide a smooth and fast user experience. The backend is handled using Firebase, which stores registration details, payment information, and admin data securely.

An important feature of the system is automated payment verification. The admin can upload the bank statement, and the system checks the payment reference details internally and updates the payment status automatically. The admin also has permission to edit participant details and download data in Excel or PDF format. This system reduces manual work and improves accuracy and efficiency in fest management.

2. Introduction

College fests are an important part of student life and usually involve many events, participants, and payments. In most colleges, fest management is still done using manual methods such as paper forms, spreadsheets, or simple online forms. These methods often lead to confusion, data errors, and difficulty in tracking payments.

This project aims to provide a digital solution for managing fest events and registrations efficiently. The Fest Management Website acts as a single platform where students can easily register for events and admins can manage everything from one place. The system is designed to be simple, user-friendly, and secure.

By using modern web technologies, this project helps reduce paperwork, save time, and make fest organization smoother and more transparent.

3. Existing System

In the existing system, fest registrations and payments are handled manually:

  • Participants fill out forms or Google Forms.
  • Payments are verified by manually checking bank statements.
  • Data is stored in Excel sheets.
  • Any correction or update requires manual effort.

This system becomes difficult to manage when the number of participants increases.

4. Limitations of the Existing System

  • Manual data entry leads to errors.
  • Payment verification takes a lot of time.
  • No real-time payment status update.
  • Data is not centrally managed.
  • High workload for organizers.
  • Risk of data loss or duplication.

5. Proposed System

  • The proposed system is a web-based Fest Management Website that automates registration and payment verification. Participants can register for events and submit payment details without seeing any internal ID. All internal matching and verification is handled only by the admin side.
  • Admins get a secure dashboard where they can manage events, view registrations, upload bank statements for verification, edit participant details, and download reports. The system is designed to be simple, secure, and efficient.

6. Features of the Proposed System

1. Event Display

  • All fest events are displayed on the website.
  • Users can view event details easily.

2. Online Registration

  • Participants can register through an online form.
  • Registration data is stored securely.
  • No unique ID is shown to users.
Fig 2: User Registration Process Flow

3. Payment Handling

  • Users make payment through UPI or bank transfer.
  • Payment reference details are collected during registration.

4. Automated Payment Verification

  • Admin uploads the bank statement (PDF).
  • System checks payment details internally using OCR and PDF parsing.
  • Payment status changes from Pending to Verified automatically.
Fig 3: Automated Admin Verification Flow

5. Admin Panel

  • Secure admin login.
  • Admin can edit participant details.
  • Admin can manage events and registrations.

6. Report Generation

  • Data can be downloaded as Excel or PDF.
  • Useful for documentation and records.
